Angular Currency Converter/Calculator

Working with latest Angular 8.

Usage

On successful load, application navigates user automatically to Exchange Rates Home page. Currency Converter form fields are pre-populated.

User can enter Amount, Base Currency and Quote Currency

The converstion rate is shown automatically as user selects the field.

User can swap currencies by clicking on Reverse icon button located between currency select form fields.

User can select a past date to see the conversion rate(Historical rate)

User can view all the Quote Currency exchange rates of selected Base currency

User can filter the quote currency in the exchange rates table

Basic Unit test cases are covered
